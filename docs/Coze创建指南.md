# Coze创建Bot指南 - 深圳学位房规划助手

## 步骤1：创建智能体
1. 打开 https://www.coze.cn/home
2. 点击"创建" → "创建智能体"
3. 基本信息：
   - 名称：深圳学位房规划助手
   - 描述：帮中介秒查学位房方案，给客户出定制规划

## 步骤2：配置人设与回复逻辑
将 `./学区房规划Agent/系统提示词.md` 的完整内容粘贴到"人设与回复逻辑"输入框

## 步骤3：添加知识库
1. 点击左侧"知识" → "知识库" → "创建知识库"
2. 创建3个文本知识库，分别上传：
   - **深圳学校梯队**：上传 `./学区房规划Agent/知识库/深圳学校梯队.md`
   - **学区房价格速查**：上传 `./学区房规划Agent/知识库/学区房价格速查.md`
   - **时间线与政策**：上传 `./学区房规划Agent/知识库/时间线与政策.md`
3. 每个知识库设置：
   - 检索策略：混合检索
   - 最小匹配度：0.5
   - 开启"查询改写"和"结果重排"

## 步骤4：添加插件
1. 点击左侧"插件" → 添加插件
2. 搜索并添加"联网搜索"插件（用于验证数据）

## 步骤5：配置开场白
```
你好！我是深圳学位房规划助手 🏫

告诉我你的情况，我帮你规划：

**快速查询**（中介模式）：
- "300万能上什么学校？"
- "龙华第一梯队初中有哪些？"
- "深中龙华录取线多少分？"

**定制方案**（客户模式）：
- 说"出方案"+ 你的预算、户籍、小孩年级
- 例如："出方案，300万，深户，小孩4年级，想上第一梯队初中"

我会在回答前联网验证数据，确保信息准确 ⚡
```

## 步骤6：发布
1. 点击右上角"发布"
2. 选择发布渠道：
   - ✅ Bot As API（供Web页面调用）
   - ✅ Web SDK（可嵌入网页）
3. 发布后记录Bot ID（URL中bot/后面的数字）

## 步骤7：获取API Token
1. 打开 https://www.coze.cn/open/oauth/pats
2. 点击"添加令牌"
3. 名称：xuequ-api
4. 权限：勾选 chat
5. 保存Token，后续Web后端需要用到

- API Token
